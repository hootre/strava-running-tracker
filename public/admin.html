🔔 알림 관리

목/금/토 오후 6시(KST)에 Vercel Cron이 자동 발송합니다.
이 버튼은 즉시 수동 발송용입니다.

📋 VAPID 키 설정 방법
  1. 터미널에서 실행: npx web-push generate-vapid-keys
  2. 출력된 Public Key → VAPID_PUBLIC_KEY 환경변수에 저장
  3. 출력된 Private Key → VAPID_PRIVATE_KEY 환경변수에 저장
  4. VAPID_EMAIL=mailto:your@email.com 설정
  5. CRON_SECRET=임의의_긴_문자열 설정 (Cron 보안)
  6. Vercel 대시보드 → Settings → Environment Variables에 추가
  7. 재배포 후 멤버들이 앱에서 🔔 버튼으로 알림 구독

1️⃣ 권한 링크 생성

멤버의 Client ID를 입력하면 권한 부여 링크가 생성됩니다. 멤버에게 이 링크를 보내주세요.

아래 링크를 멤버에게 보내주세요:

멤버가 링크를 열고 승인하면 화면에 code=... 가 표시됩니다. 그 코드를 받아서 아래 2단계에 입력하세요.

2️⃣ 멤버 등록

📋 전체 흐름

  1. 멤버가 strava.com/settings/api 에서 API 앱 생성
  2. 멤버가 Client ID와 Client Secret을 대표님한테 전달
  3. 대표님이 위 1단계에서 권한 링크 생성 → 멤버에게 전송
  4. 멤버가 링크 열고 "승인" 클릭 → 화면에 나온 코드를 대표님한테 전달
  5. 대표님이 2단계에서 이름 + Client ID + Secret + 코드 입력 → 등록 완료!
